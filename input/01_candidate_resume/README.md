本目录用于收纳候选人简历和候选人相关输入资料，供独立的人才评估线程读取和判断。

## 目录角色

用于把候选人资料与 Kinbot 当前团队规划基线连接起来，支撑“是否符合当前组织能力缺口”的评估。

## 文档索引

当前暂无已纳入版本维护的候选人简历文件。

## 使用边界

1. 候选人评估线程应以 `docs/10_team_planning/01_development_team_proposal.md` 作为团队能力基线。
2. 评估时应优先判断候选人是否匹配当前最急缺岗位、阶段扩编重点和 `S1-S7` 或横向团队职责。
3. 本目录用于保存输入资料，不在本目录内直接冻结组织结论；正式判断应回写到独立评估线程或其输出文档。
