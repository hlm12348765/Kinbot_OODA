## 1. 目录定位

本目录用于承接：

1. `03_decision_log.md` 拆出的历史决策分卷；
2. 面向当前协作的派生索引；
3. 只服务导航与检索的辅助入口。

规则：

1. `03_decision_log.md` 继续是唯一正式总入口。
2. 本目录中的派生索引不构成新的并列事实源，只用于帮助阅读和定位。
3. 历史决策仍按时间分卷，不按专题拆成多个平级主文档。
